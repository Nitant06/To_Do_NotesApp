# To_Do_NotesApp

It allows users to make a list of daily task in the form of To-Do List and store the list locally in their device using SQLite database.

Concepts used in implementing the project are:

1. MVVM Architecture

2. Room Database

3. SQLite

4. DAO

5. Repository

6. ViewModel

7. LiveData

8. RecyclerView
